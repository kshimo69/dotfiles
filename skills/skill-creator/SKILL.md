# スキル作成ガイド

## スキル作成プロセス

1. 具体例でスキルを理解する
2. 再利用可能なコンテンツを計画（スクリプト、参照、アセット）
3. スキルを初期化（init_skill.py を実行）
4. スキルを編集（リソース実装と SKILL.md 作成）
5. スキルをパッケージ化（package_skill.py を実行）
